# Text Adventure

This simple project is a small game use to discover Python language

## Installation

```bash
git clone https://github.com/ThT12/TextAdventure.git
cd TextAdventure
python setup.py install
```

## Usage

Launch the game using:

```bash
play_text_adventure
```

## Unit Tests

```bash
python -m unittest
```
